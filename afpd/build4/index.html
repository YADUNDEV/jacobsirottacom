Abomination Fight Pit Deluxe 0.3.1

Gameplay Footage

About

0.3.1 is where I consider the game to have reached the level of polish of a full game. The game could now be played with up to 4 players, players can now configure their gamepad controls however they like allowing support for more input devices. A main menu was added, ui was improved, and bugs were squashed.

Original Patch Notes

Big Additions

  • You can now battle with up to 4 players!
    • Select the two arrows at the bottom of the CSS to adjust the number of players.
  • Team battles have been added!
    • Select the mode option at the top of the CSS to swap modes when more than two players are playing.
    • Select the flag button to swap teams.
  • You can now bind custom controls!
    • Select the controller icon at the bottom of the CSS to do so. These settings will persist until the page is refreshed.
  • You can now set a time limit for matches!
  • A main menu with additional options has been added!

Other Notable Additions

  • Added a salty runback feature: hold the special attack button at the end of a match to replay that match with the same settings.
  • All players now have control over every menu! Now you can fight before you fight!

Balance tweaks

  • Sputnik
    • Sputnik can now jump out of GAF starting on frame 6.
    • Boosted the speed gained from GAF from 350 to 450.
    • Decreased Sputnik's floatiness factor (weight) from 1.3 to 1.15. This will make him slightly harder to KO.
  • Prince Slime
    • Decreased damage on Prince Slime's grabs from 5 to 3 (The grab portion, not the throw).
    • Increased Prince Slime's AAF endlag from 7 frames to 11 frames.
    • Decreased Prince Slime's GAF endlag from 22 frames to 12 frames.
    • Increased the time you can jump out of GAD from the first 3 frames to the first 4.

Other Notable Gameplay Changes

  • Adjusted Failure City's blast zone.
  • Intangibility from an air dash now ends after using an attack.
  • Fast falling now increases fall acceleration by a factor of 2.
  • You can now cancel a fast fall by double jumping.
  • Jumping out of an attack that allows it will now also cancel hitstop, making it easier to perform.
  • Default gamepad axis sensitivity increased to 1.1. (This can be adjusted in settings.)

Other Notable Audio/Visual Changes

  • Implemented attack whiff sound effects.
  • Character sounds' volume will now scale based on distance from the center of the scree. (This is all I can do atm, Phaser doesn't support audio panning lol).
  • Many sound effects now have a slightly randomized pitch.
  • Cursors now have a unique graphic for hovering over a selectable option.

Notable Bug Fixes

  • Major camera improvements; Camera will no longer spaz out sometimes.
  • Cats no longer have an infinite hitbox while stationary on platforms.
  • Throwing an item will no longer put out a hitbox at the position where it was picked up for 1 frame after throwing.
  • On hit particle effects are now positioned correctly.
  • Character item grip positions now update correctly.
    • (Note that Sputnik and Prince Slime use placeholder item grip positions)
  • Fixed an issue where horizontal knockback was set to zero when players were at the same exact x pixel position (Yes, I actually somehow ran into this while testing and had to debug it.)


Goals For Next Patch

  • Add wall spikes.
  • Improve collision stability.
  • Add a new character.
